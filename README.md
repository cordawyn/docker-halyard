# docker-halyard

Docker image for [Halyard](https://github.com/merck/halyard) triplestore.

# How to use

## Deploying HBase

To deploy HBase use official BDE project [HBase docker image](https://github.com/big-data-europe/docker-hbase).

## Using Halyard SDK

```
make run-sdk
```

## Using Halyard Webapps

```
make run-webapps
```
